# healthmodelling

Dataset Overview

This dataset pools data from three independent studies:
	•	SCANS
	•	RUN DMC
	•	ASPS

The pooled dataset is used to examine the prediction of dementia using:
	•	a Simple MRI SVD score
	•	an Amended MRI SVD score

⸻

Outcome Variable

Dementia During Follow-up
	•	Variable: Dementia
	•	Coding:
	•	1 = Dementia
	•	0 = No dementia
